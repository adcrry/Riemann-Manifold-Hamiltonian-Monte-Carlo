# Riemannian Hamiltonian Monte Carlo (RMHMC)

This repository contains a Python implementation of Riemannian Hamiltonian Monte Carlo, aiming to reproduce results from relevant literature.

## Project Structure

- `src/rmhmc`: Source code for the RMHMC implementation.
- `tests`: Unit tests.

## Installation

```bash
pip install -e .
```

## Goals

- Implement the Riemannian Manifold Hamiltonian Monte Carlo algorithm.
- Reproduce experiments and figures from the target paper.
